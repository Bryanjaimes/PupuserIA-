# Blockchain Contracts — Foundation Fund Tracking
#
# Smart contracts for transparent, on-chain tracking of
# Foundation fund allocations. Every dollar is traceable
# from platform transaction to child impact.
#
# Target chains: Stellar (primary) or Polygon (alternative)
#
# Contract types:
#   - FundAllocation: Records revenue → Foundation transfers
#   - ProgramDelivery: Records fund → program deployments
#   - ImpactVerification: On-chain attestation of impact delivery
